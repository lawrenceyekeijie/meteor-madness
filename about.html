This project demonstrates asteroid impact scenarios using real-world data from NASA and USGS. It aims to educate scientists, policymakers, and the public about asteroid risks and possible mitigation strategies.

Features

  • 🌍 Visualize asteroid impact zones on an interactive world map
  • 📊 Simulate asteroid parameters (mass, velocity, location, mitigation)
  • 📡 Fetch country data for population impact estimates
  • ☄️ Explore potential strategies for asteroid deflection
